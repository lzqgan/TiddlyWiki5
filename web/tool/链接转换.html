格式说明

普通

站点名称,网址

示例: 谷歌搜索,https://www.google.com

Markdown

[站点名称](网址)

示例: [谷歌搜索](https://www.google.com)

太微

[[站点名称|网址]]

示例: [[谷歌搜索|https://www.google.com]]

DokuWiki

[[网址|站点名称]]

示例: [[https://www.google.com|谷歌搜索]]

OneTab

网址|站点名称

示例: https://www.google.com|谷歌搜索

HTML

<a href="网址">站点名称</a>

示例: <a href="https://www.google.com">谷歌搜索</a>
